Algorithm Pipeline

1. Color Space Shift: The RGB image is converted to YCbCr. This isolates details on the Y (Luminance) channel. Humans are highly sensitive to Y details, so the SRCNN network runs solely on Y, while CbCr are bilinearly upsampled.

2. Bilateral Denoising: A bilateral filter smooths out flat surfaces, suppressing sensor noise and JPEG artifacts without destroying crucial edges.

3. Feature Extraction (Layer 1): Convolves the upscaled Y image with 4 distinct 5x5 filters. Each filter extracts a specific class of high-frequency feature (horizontal edges, vertical edges, local curvature, ridge textures) using a PReLU activation.

4. Non-Linear Mapping (Layer 2): Interconnects the 4 feature maps spatially (3x3 neighborhood) and cross-channel via a coupling matrix, creating abstract patterns.

5. Reconstruction & Sharpening (Layer 3): A 5x5 convolution recombines the mapping grids to generate a high-frequency residual detail map. This is blended back to the bicubic upscaled luminance. Finally, adaptive unsharp masking guided by Sobel edge detection sharpens boundaries while suppressing noise amplification.
